Velocity Run

  • VELOCITY RUN is a 3D real-time car racing and endless runner game.
  • Developed using C++, OpenGL, and GLUT libraries.
  • Demonstrates computer graphics concepts such as 3D rendering, transformations, lighting, and animations.
  • Features user interaction, collision detection, and environmental effects.
  • Includes diverse maps, obstacles, and collectible objects.
  • Implements advanced graphics effects such as fog, weather simulation, terrain and water rendering.
  • Uses HUD (Head-Up Display) mechanisms to enhance gameplay.
  • Applies theoretical computer graphics concepts in an interactive gaming application.

Requirements Language: C++
Graphics Library: OpenGL / GLUT
IDE Context: Developed and configured using Code::Blocks (includes .cbp project and layout setup).
